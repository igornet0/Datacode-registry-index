# Индекс реестра библиотек Datacode

Этот репозиторий хранит **связи (метаданные)** на пакеты библиотек для Datacode: имена, источники (URL репозиториев), описания и требования к версии Datacode. Сам код библиотек здесь не лежит — только индекс, по которому инструменты могут находить и подключать пакеты.

**Документация также доступна на [английском](README_EN.md).**

## Файл `config.json`

Корневой файл индекса — `config.json`. Структура:

| Поле | Описание |
|------|----------|
| `version` | Версия схемы индекса (целое число). |
| `packages` | Массив записей о пакетах. |

Каждый элемент `packages` может содержать:

| Поле | Описание |
|------|----------|
| `name` | Имя пакета (идентификатор в экосистеме Datacode). |
| `source` | URI источника, обычно Git-URL в формате `git+https://...`. |
| `description` | Краткое описание и при необходимости шаги сборки или размещения артефактов. |
| `min_datacode` | Минимальная совместимая версия Datacode (например, `>=2.0.0`). |

## Как добавить или изменить пакет

1. Отредактируйте `config.json`: добавьте объект в `packages` или обновите существующий.
2. Сохраните осмысленное описание в `description`, чтобы пользователям было понятно, как собрать или подключить библиотеку.
3. Отправьте изменения через pull request в этот репозиторий.

## Лицензия и вклад

Содержимое индекса предназначено для свободного использования в связке с Datacode. Условия распространения самих библиотек смотрите в их репозиториях.
